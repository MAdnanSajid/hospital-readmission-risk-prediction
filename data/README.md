## Dataset Description

This project uses publicly available healthcare data published by UK government organisations.

### Data Source
- NHS Digital: https://digital.nhs.uk/data-and-information
- Office for National Statistics (ONS): https://www.ons.gov.uk/

### Data Collection
The data was originally collected by NHS hospitals across the United Kingdom during routine healthcare service delivery and administrative processes. It is aggregated and published by NHS Digital and the Office for National Statistics for research and public use.

### Data Type
The dataset is tabular in nature and includes:
- Categorical variables (e.g. gender, admission type, diagnosis category)
- Numerical/continuous variables (e.g. age, length of stay, number of previous admissions)

### Notes
Raw data will be stored in this directory without modification. Any cleaned or transformed versions of the data will be saved separately to ensure reproducibility.
